Modo treino adaptado para o jogo World Heroes Perfect. Usa o emulador MAME, **(já incluso)** para iniciar o jogo execute o arquivo whp.bat.

![World Heroes Perfect](https://the-gamers.github.io/trainingmode/images/whp.jpg)

* Ao iniciar o jogo vai diretamente para a tela de player select 
* O personagem para o Player 2 é selecionado automaticamente
* Marcador de combos e danos fixo na tela
* Para retornar a tela de player select mantenha **START** pressionado por alguns segundos
* Para fechar o jogo pressione tecla **ESC**

Download disponível [aqui](https://drive.google.com/file/d/1ApXp66p6gfccd3zGS7I3zzsCbxezb9w1/view?usp=sharing)
