Modo treino adaptado para o jogo Real Bout Fatal Fury 2. Usa o emulador MAME, **(já incluso)** para iniciar o jogo execute o arquivo start.bat.

![Real Bout Fatal Fury 2 - The New Comers](https://the-gamers.github.io/trainingmode/images/rbff2.png)

* Ao iniciar o jogo vai diretamente para a tela de player select
* É possível escolher o personagem do Player 2
* Alfred selecionável pressionando **A+C**
* Marcador de combos e danos fixo na tela
* Para retornar a tela de player select mantenha **START** pressionado por alguns segundos
* Para fechar o jogo pressione tecla **ESC**

Download disponível [aqui](http://j.gs/25197835/rb2)
