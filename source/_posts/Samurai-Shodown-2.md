Modo treino adaptado para o Samurai Shodown II. Usa o emulador MAME, **(já incluso)** para iniciar o jogo execute o arquivo start.bat.
 
![Samurai Shodown 2](https://the-gamers.github.io/trainingmode/images/ss2.jpg)

* Ao iniciar o jogo vai diretamente para a tela de player select
* O personagem do Player 2 é selecionado automaticamente
* Marcador de danos fixo na tela
* Para retornar a tela de player select mantenha **START** pressionado por alguns segundos
* Para fechar o jogo pressione tecla **ESC**
