Modo treino adaptado para o jogo Real Bout Fatal Fury. Usa o emulador MAME, **(já incluso)** para iniciar o jogo execute o arquivo start.bat.


![Real Bout Fatal Fury](https://the-gamers.github.io/trainingmode/images/rb.jpg)

* Ao iniciar o jogo vai diretamente para a tela de player select
* É possível escolher o personagem para o Player 2
* Marcador de combos, max combo e danos fixo na tela
* Para retornar a tela de player select mantenha **START ** pressionado por alguns segundos
* Para fechar o jogo pressione tecla **ESC**

Download disponível [aqui](http://j.gs/25197835/realbout)
